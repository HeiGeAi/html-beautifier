RELEASE RULES

正式版输出的硬约束

以下 12 条规则来自 v1.3.0 的 SKILL.md 和验收清单，示例页本身也按这些限制实现。

必须满足（6 条）

  • 单文件 HTML，样式全部内联在 <style> 中
  • 默认可见，无需点击、滚动或等待动画
  • 仅依赖 Google Fonts 和 Font Awesome CDN
  • 桌面、平板、手机都有基础响应式回落
  • 单张卡片至少承载 3 条有效信息（v1.3.0 新增）
  • grid gap ≤ card padding，避免密度倒挂（v1.3.0 新增）

明确禁止（6 条）

  • 任何 animation / transition / hover 效果
  • JavaScript 交互逻辑和默认隐藏内容
  • Inter / Roboto / Arial 等通用字体
  • emoji、破折号等 AI 味元素
  • 卡片只装 1 条信息、空白撑高卡片（v1.3.0 新增）
  • section 外壳大 padding 套 card 大 padding（v1.3.0 新增）
