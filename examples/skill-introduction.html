核心特性

三种布局模式

Editorial

长滚动阅读，适合报告、分析、方案展示。章节清晰，打印友好。

Dashboard

看板式布局，适合数据汇总、状态监控。信息密度高，一屏展示。

Single

单页海报式，适合活动公告、单页简介。视觉冲击力强。

核心组件

  • card-light - 白底卡片，万能信息容器
  • gt-渐变文字 - 标题强调词渐变色
  • section-divider - 章节标题下方渐变短线
  • badge - 药丸标签，分类标识
  • num-dot - 编号圆点，列表序号
  • icon-box - 方形圆角图标框

设计理念

「卡片即信息容器，留白即呼吸感，节奏即叙事力」

所有内容通过精心设计的卡片、标签、图标组合来承载，不堆砌文字。每一个模块都是一张完整的视觉海报，有清晰的信息层次。

输出特点

单文件输出

所有 CSS 内联，零依赖（仅 CDN 字体和图标）

打印友好

无动画效果，打开即为最终呈现效果

响应式设计

支持移动端，自适应不同屏幕尺寸

所见即所得

静态页面，无需 JavaScript，加载即可查看
